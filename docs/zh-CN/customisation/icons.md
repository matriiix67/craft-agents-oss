# Icons（图标）

> 原文：https://agents.craft.do/docs/customisation/icons.md
>
> 本文是中文摘译版，说明 sources、skills 和 statuses 的图标配置。

## 概览

Icons 用来提高列表和 sidebar 的可扫描性。你可以为 sources、skills、statuses 等实体配置图标，让不同类型内容更容易识别。

## 可配置对象

* Sources
* Skills
* Statuses
* Labels 或 smart labels
* Workflow entries

## 图标选择建议

* 图标语义要直观
* 同类对象使用一致风格
* 避免过度装饰
* 小尺寸下仍能辨认
* 与颜色搭配时保持对比度

## 示例

| 对象 | 图标语义 |
| ---- | -------- |
| GitHub source | repository、branch、code |
| Linear source | issue、list、workflow |
| Docs skill | document、file-text |
| Review status | eye、check-circle |
| Blocked status | alert、stop |

## 维护建议

图标是识别系统的一部分。调整前先确认团队是否已经形成使用习惯，避免频繁修改导致扫描成本上升。
