# Themes（主题）

> 原文：https://agents.craft.do/docs/customisation/themes.md
>
> 本文是中文摘译版，说明 Craft Agents 的主题系统。

## 概览

Themes 用来定制 Craft Agents 的视觉外观。主题通常基于一组语义颜色，覆盖背景、前景、强调色、边框和状态显示。

## 主题可以控制什么

* 应用整体色调
* 明暗模式表现
* Sidebar 和面板背景
* 文字、边框和强调色
* Labels、statuses 和 badges 的视觉协调

## 6 色系统

主题通常围绕少量核心语义颜色构建。保持颜色数量有限，可以让界面稳定、可读，也便于维护。

## Workspace 级主题

不同 workspace 可以使用不同视觉风格，帮助快速区分上下文。例如个人 notes、工作项目和远程 server 可以使用不同主题。

## 编写建议

* 优先保证可读性和对比度
* 暗色模式和亮色模式都要检查
* 避免把 status color 和主题强调色混淆
* 保持背景层级清晰

## 维护建议

主题是长期配置。修改前可以先复制现有主题，再小范围调整颜色，确认 UI 中列表、按钮、输入框和 overlay 都能正常阅读。
